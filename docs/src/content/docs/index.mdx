**harnessapi** is a Python framework that turns a skill folder into a **streaming HTTP API** and a **Model Context Protocol (MCP) tool** — simultaneously, automatically, with zero boilerplate.

```bash
# Try it instantly — no install needed
uvx harnessapi init my-project
cd my-project
uvx harnessapi run
```

```bash
curl -X POST http://localhost:8000/skills/greet \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"name": "world"}'
# → {"message": "Hello, world! Welcome to harnessapi.", "length": 36}
```

## What you get from one skill folder

<CardGrid>
  <Card title="Streaming HTTP endpoint" icon="rocket">
    `POST /skills/{name}` — Server-Sent Events by default, plain JSON via `Accept` header. No routes to write.
  </Card>
  <Card title="MCP tool" icon="puzzle">
    Every skill is automatically registered as a Model Context Protocol tool at `/mcp`. Plug into Claude Desktop, Cursor, or any agent in seconds.
  </Card>
  <Card title="OpenAPI docs" icon="document">
    Full Swagger UI at `/docs` — generated from your Pydantic models. Zero extra configuration.
  </Card>
  <Card title="agentskills.io compatible" icon="approve-check">
    Existing skill folders with `SKILL.md` are detected automatically. harnessapi is a superset of the agentskills.io standard.
  </Card>
</CardGrid>

## Built on

- [FastMCP](https://github.com/jlowin/fastmcp) — the MCP server
- [Pydantic](https://docs.pydantic.dev) — input/output validation
- [uv](https://docs.astral.sh/uv/) — packaging and CLI
